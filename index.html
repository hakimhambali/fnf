How to Format Your Name for Flight Bookings

Airlines require your name to match your passport exactly. Incorrect names may result in denied boarding or additional fees for name corrections. This tool helps you split your full name into the correct fields used by each airline.

Malaysia Airlines (MAS)

Malaysia Airlines uses two fields: First & Middle Name and Last Name / Surname.

  • Malay names: Place the given name in "First & Middle Name" and Bin/Binti/Anak plus the patronymic in "Last Name / Surname". Example: AHMAD FALIQ BIN HAMEDI → First & Middle Name: AHMAD FALIQ, Last Name: BIN HAMEDI.
  • Chinese names: Place the family name in "Last Name / Surname" and the remaining names in "First & Middle Name". Example: TAN MEE LING → First & Middle Name: MEE LING, Last Name: TAN.
  • Indian names: Place the given name in "First & Middle Name" and the family or patronymic name in "Last Name / Surname".

Reference: Malaysia Airlines Booking Guide

AirAsia

AirAsia uses two fields: Given Name and Family Name / Surname. The formatting logic is the same as Malaysia Airlines.

  • Malay names: Given Name: AHMAD FALIQ, Family Name: BIN HAMEDI.
  • Chinese names: Given Name: MEE LING, Family Name: TAN.
  • Remove the @ symbol and replace with a space. Omit A/P or A/L unless they appear in the name section of your passport. Replace hyphens with a space.

Reference: AirAsia Guest Details Guide

Batik Air

Batik Air uses two fields: First Name and Last Name. Unlike other airlines, Bin/Binti is dropped from the name.

  • Malay names: Drop Bin/Binti and place only the father's name in Last Name. Example: MOHAMMED FAZIL BIN MOHAMMED FALEEL → First Name: MOHAMMED FAZIL, Last Name: MOHAMMED FALEEL.
  • Chinese names: Family name goes to Last Name, remaining names to First Name. Example: CATHERINE LOO SIANG YEW → First Name: CATHERINE SIANG YEW, Last Name: LOO.
  • Indian names: Given name in First Name, family name in Last Name. Example: SANDRAMOGAN MUNIANDY → First Name: SANDRAMOGAN, Last Name: MUNIANDY.
  • Single-word names: Enter the same name in both First Name and Last Name. Example: ISKANDAR → First Name: ISKANDAR, Last Name: ISKANDAR.
